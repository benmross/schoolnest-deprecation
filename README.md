# SchoolNest Domain Deprecation Notice

A simple static site that displays a deprecation notice for the schoolnest.org domain and redirects users to sparrows.school after 10 seconds.

## Purpose

This site is deployed to Vercel and serves the schoolnest.org domain to inform users that the domain has been deprecated and redirect them to the new domain at sparrows.school.

## Features

- Clean, professional deprecation notice
- 10-second countdown timer
- Automatic redirect to sparrows.school
- Skip button to redirect immediately
- Fully responsive design

## Deployment

Deploy this site to Vercel:

1. Push this repository to GitHub
2. Import the repository to Vercel
3. Add the domain `schoolnest.org` (and `www.schoolnest.org`) to the Vercel project
4. Configure DNS to point to Vercel

## Files

- `index.html` - The deprecation notice page (self-contained HTML/CSS/JS)
- `vercel.json` - Vercel deployment configuration
- `README.md` - This file
